# Preferences
## Animation
- ALL major animations: JavaScript frameAnimation.ts at 12fps
- CSS only for minor hover effects
- Stop-motion = discrete frame swaps, never smooth

## Design
- Handmade/paper aesthetic
- Elements look scattered, not perfectly aligned
- Envelopes have random rotations
- Stickers are ALWAYS buttons, never decorative

## Code
- No React, no Tailwind
- Images in public/images/
- PNGs for transparency, JPGs for backgrounds
